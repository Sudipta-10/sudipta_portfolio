# Sudipta Pratiher - Personal Portfolio

Welcome to the source code for my personal portfolio website! This project is a modern, fully responsive web application designed to showcase my skills, experience, and projects as a software developer.

### 🌐 [Live Demo: View Portfolio Here](https://sudipta-10.github.io/sudipta_portfolio/)

## 🚀 Features

- **Modern UI/UX**: Clean, minimalist design with a premium aesthetic.
- **Stacked Card Scrolling**: Features a custom, hardware-accelerated "sticky" scrolling animation where sections seamlessly slide over one another like a deck of cards.
- **Glassmorphism Design**: Custom frosted glass effects on the navigation bar.
- **Custom Branding**: Digitized, hand-drawn digital signature logo seamlessly integrated into the navigation and footer.
- **Responsive Layout**: Fully optimized for desktops, tablets, and mobile devices.
- **Interactive Elements**: Dynamic hover states, smooth scroll-spy navigation, and scroll-reveal animations.

## 🛠️ Technologies Used

- **HTML5**: Semantic markup for accessible and SEO-friendly structure.
- **CSS3**: Custom styling, CSS Grid/Flexbox layouts, CSS variables for theming, and advanced animations (backdrop-filter, sticky positioning).
- **JavaScript (Vanilla)**: DOM manipulation, custom scroll interception logic, Intersection Observers for reveal animations, and responsive navigation controls.
- **FontAwesome**: Scalable vector icons.
- **Google Fonts**: Custom typography (`Outfit` and `Dancing Script`).

## ⚙️ Getting Started

To run this project locally, you simply need to serve the static files.

### Option 1: Live Server (VS Code)
1. Open the project folder in Visual Studio Code.
2. Install the **Live Server** extension.
3. Right-click on `index.html` and select **"Open with Live Server"**.

### Option 2: Node.js / NPM
If you have Node.js installed, you can use any static server package.
```bash
# Install a simple static server
npm install -g serve

# Run the server in the project directory
serve .
```

### Option 3: Python
If you have Python installed, you can run a local server natively.
```bash
# For Python 3
python -m http.server 8000
```
Then visit `http://localhost:8000` in your browser.
